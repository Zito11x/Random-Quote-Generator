## Описание программы
Random Quote Generator — это графическое приложение на Python, которое генерирует случайные цитаты из заранее заданного списка.

Пример:

Программа позволяет:

- получать случайную цитату
- просматривать историю сгенерированных цитат
- фильтровать цитаты по автору и теме
- сохранять историю в файл JSON
- загружать историю при повторном запуске

--- 

## Требования для запуска
- Python 3.x
- (если есть) библиотеки: pip install ...

---

## Как запустить

```bash
git clone https://github.com/Zito11x/Random-Quote-Generator
cd random-quote-generator
python main.py
```

---

## Как пользоваться
1. Запустите программу
2. Нажмите кнопку «Сгенерировать цитату»
3. На экране появится случайная цитата
4. Все цитаты сохраняются в истории
5. Используйте фильтры:
   - по автору
   - по теме
6. История автоматически сохраняется в файл "history.json"

---

## Пример работы

Пример сгенерированной цитаты:

«"Stay hungry, stay foolish."
— Steve Jobs»

История:

- "Stay hungry, stay foolish." — Steve Jobs
- "To be or not to be." — William Shakespeare
